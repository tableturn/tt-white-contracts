# Solidity API

## IIssuerEvents

### FastRegistered

```solidity
event FastRegistered(address fast)
```

Emited when a new FAST is registered.

| Name | Type | Description |
| ---- | ---- | ----------- |
| fast | address | The address of the newly registered FAST diamond. |

### MemberRemoved
